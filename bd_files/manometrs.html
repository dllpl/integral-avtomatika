  По назначениям манометры можно разделить на следующие виды манометров: на технический (общетехнический) манометр, электроконтактный манометр, специальный манометр, железнодорожный манометр, виброустойчивый (глицеринозаполненый) манометр, судовый манометр и эталонный (образцовый) манометр, дифференциальный манометр для всех отраслей.
 
Общетехнический манометр - предназначен для измерения не агрессивных к сплавам меди жидкостей, газов и паров, манометры для воды, манометры для ЖКХ.
 
Электроконтактный манометр - имеет возможность регулировки измеряемой среды, благодаря наличию электроконтактного механизма.

Виброустойчивый (глицеринозаполненый, жидконаполненный) манометр - глицеринозаполненые, жидконаполненные манометры служат для увеличения срока работы манометра и увеличения низкого температурного порока, обеспечивают отсутствия вибрации стрелки манометра.

Манометры из нержавеющей стали – предназначены манометры для использования с агрессивными средами и являются манометрами для аммиака, аммиачный манометр.

Дифференциальный манометр – это дифференциальные манометры, которые позволяют измерить дифференциальное давление дифференциальным манометром. Мы делаем дифференциальные манометры двух видов – мембранные дифференциальные манометры, дифференциальные манометры с магнитным поршнем для различным областей промышленности.
